# student-performance-analysi
Analiza czynników wpływających na wyniki studentów. Projekt obejmuje czyszczenie danych, analizę eksploracyjną (EDA), wizualizację danych oraz model regresji. Identyfikacja zależności między wynikami a nauką, frekwencją i czynnikami społecznymi.
# Student Performance Analysis

Analiza czynników wpływających na wyniki studentów.

## Zakres:
- czyszczenie danych
- analiza eksploracyjna (EDA)
- wizualizacja danych
- analiza korelacji
- model regresji

## Wnioski:
Wyniki zależą głównie od liczby godzin nauki oraz wcześniejszych osiągnięć.
