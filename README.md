# Student Performance Analysis

Projekt analizy czynników wpływających na wyniki studentów.

## Zakres:
- czyszczenie danych
- analiza eksploracyjna (EDA)
- wizualizacja danych
- analiza korelacji
- model regresji

## Wyniki:
Największy wpływ na wynik egzaminu mają:
- liczba godzin nauki
- wcześniejsze wyniki
- frekwencja
